Apply percentage-based discounts to WooCommerce products by category with automatic strikethrough pricing display.

== Description ==

WooCommerce Category Discount allows you to easily set percentage-based discounts for entire product categories. Perfect for running category-wide sales without manually editing each product.

= Features =

* **Easy Setup** - Set discounts directly from WooCommerce admin
* **Percentage Discounts** - Apply 0-100% discounts to any category
* **Visual Pricing** - Automatic strikethrough original price with sale price
* **Discount Badge** - Shows discount percentage on product pages
* **Smart Discounting** - Applies highest discount when product is in multiple categories
* **Variable Products** - Full support for variable products with price ranges
* **Cart Integration** - Discounted prices reflected in cart and checkout
* **Translation Ready** - Fully internationalized with POT file included

= How It Works =

1. Navigate to WooCommerce → Category Discounts
2. Set a percentage discount for any product category
3. Products automatically display with strikethrough pricing and discount badge

= Requirements =

* WordPress 5.8 or higher
* WooCommerce 5.0 or higher
* PHP 7.4 or higher

== Installation ==

= Automatic Installation =

1. Log in to your WordPress dashboard
2. Navigate to Plugins → Add New
3. Search for "WooCommerce Category Discount"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Log in to your WordPress dashboard
3. Navigate to Plugins → Add New → Upload Plugin
4. Choose the zip file and click "Install Now"
5. Activate the plugin

= Configuration =

1. Go to WooCommerce → Category Discounts
2. Enter discount percentages for desired categories
3. Click "Save Discounts"

== Frequently Asked Questions ==

= Does this work with variable products? =

Yes! Variable products fully support category discounts. The discount is applied to all variations and the price range is displayed with strikethrough.

= What happens if a product is in multiple categories with different discounts? =

The plugin automatically applies the highest discount percentage from all categories the product belongs to.

= Does this affect the cart and checkout prices? =

Yes, the discounted prices are applied throughout the entire shopping experience including cart, checkout, and order totals.

= Can I set different discounts for subcategories? =

Yes, each category (including subcategories) can have its own discount percentage.

= Will this conflict with existing sale prices? =

The category discount replaces any existing sale price with the category-based discount. The original regular price is preserved.

= Is this compatible with other WooCommerce plugins? =

The plugin uses standard WooCommerce hooks and filters, making it compatible with most themes and plugins. If you encounter issues, please report them in the support forum.
